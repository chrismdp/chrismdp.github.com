Image Generation Benchmark Report
11 February 2026 · 4 tests · 8 images

Executive Summary

Qwen-Image-2.0 was announced on 10 February 2026 by Alibaba's Qwen team, claiming superior text rendering and native 2K output. The model is currently in invitation-only beta; the publicly available qwen-image-max on DashScope was tested instead. This report compares it head-to-head with Gemini nano-banana-pro-preview (the model behind the /gemini-images skill) across four use cases matching the existing image generation templates.

Gemini wins 3–1 overall. Text accuracy, style adherence, and resolution make it the clear choice for the current workflow.

Conclusions & Recommendations

Where Gemini excels

  • Text accuracy — Zero errors across all 4 tests. Qwen had typos in 2 of 4 (infographic and quote card), which is a dealbreaker for professional content.
  • Style adherence — Consistently delivers the hand-drawn colored-pencil aesthetic that the brand templates specify. Qwen defaults to a more photorealistic digital illustration style.
  • Resolution — Native 2K output (up to 2304px tall for infographics). Qwen caps at 1664×928, which can't meet the infographic template spec of 1856×2304.
  • Creative interpretation — Added the superhero cape in the comic, the rubbish bin for "killed projects" in the infographic. More inventive with the brief.

Where Qwen excels

  • Data visualisation — The presentation slide S-curve was genuinely better composed, with dot clusters that communicate density changes intuitively.
  • File weight — Consistently higher-resolution output files (1.6–2.5 MB vs Gemini's 0.1–1.6 MB), suggesting more detail per pixel.
  • Price — $0.075/image vs Gemini's usage-based pricing. 100 free images on signup.

Key limitations of Qwen-Image-Max

  • Max resolution 1664×928 — Can't produce the 1856×2304 infographics or 1920×1080 presentation slides at native resolution.
  • Text rendering errors — Despite marketing claims about superior text handling, produced "ausks", "juss", "dalloring", and "chrasmedp.com" in testing.
  • Style rigidity — Tends toward photorealistic rather than adapting to the requested hand-drawn aesthetic.
  • No inline image return — Returns a temporary URL (expires in 24h) rather than base64 data, adding complexity to automation.

Recommendation

Stick with Gemini for the /gemini-images skill. Qwen-Image-Max is not a viable replacement for any of the current template types. The text accuracy problems alone are disqualifying — you can't publish infographics with "dalloring" instead of "delivering".

Worth revisiting when Qwen-Image-2.0 gets a public API (currently invitation-only on Bailian). The 2.0 model claims 2048×2048 native output and improved text rendering, which could address the two biggest gaps. Monitor the DashScope model list for a qwen-image-2.0 or updated qwen-image-max model name.

One possible niche use: If you ever need chart/diagram-style slides where data visualisation matters more than brand aesthetics, Qwen could serve as a secondary option.
